# LeadUp Asset Use Cases

Real-project examples so the skill recommends assets that fit the
LeadUp portfolio. Use these as defaults when the user names the
project; override only when the brief explicitly says so.

---

## Hostendor (hosting / server platform)

**Brand vibe:** clean, technical, trustworthy. Cool blues, teal, dark
backgrounds.

**Typical assets needed:**
- Hero photos of server rooms / data centers (Unsplash, Pexels) — but
  prefer **illustrations** for an ownable feel.
- Server / DNS / cloud illustrations (unDraw, Storyset "server"
  category, SVG Repo for cloud icons).
- Icons for DNS, SSL, CDN, uptime, backups — Lucide / Tabler / Phosphor
  (line set).
- Lottie animations for "server status", "deploying", "secured" —
  LottieFiles "Free + Commercial" filter.
- 3D icons for hosting plans / pricing cards — IconScout free 3D, Spline
  Community.
- Video clips: data-center b-roll loops for hero (Pexels Videos,
  Coverr) — keep under 1.5 MB WebM.

**Avoid:** clichéd "hooded hacker" stock; AI-rendered fake server rooms
without context.

## LeadUp website (agency)

**Brand vibe:** modern, clean, trustworthy, business-but-friendly.
Neutral palette with a brand accent.

**Typical assets needed:**
- Agency / team photos — prefer real photos by an actual photographer
  (out of scope for free finder; flag this). For placeholders, use
  Unsplash "office / team / meeting" with people in candid shots.
- SaaS / web-build illustrations — unDraw, Storyset.
- Icons for services (Web, SaaS, Apps, Hosting, Marketing) — Lucide /
  Tabler.
- Process / workflow Lottie animations.
- Testimonial avatar placeholders — DiceBear (initials / lorelei),
  Boring Avatars.

**Flag:** model release for any Unsplash team photo used on hero / ad
creatives; suggest commissioning real photography for the final site.

## Jewellery SaaS (billing / inventory for jewellery shops)

**Brand vibe:** premium, warm, gold accent, trustworthy.

**Typical assets needed:**
- Jewellery photos — Unsplash / Pexels "gold jewellery", "necklace",
  "ring" with simple backgrounds.
- Billing / inventory illustrations — Storyset (attribution flag),
  unDraw.
- Icons for billing, inventory, GST, customer, reports — Lucide /
  Tabler.
- Gold-themed vectors / patterns — SVG Repo (filter to MIT / CC0).
- Customer avatar placeholders — DiceBear (initials).

**Avoid:** generic "money" stock photos; recognisable real brand
jewellery designs (trademark / IP risk).

## Salon SaaS (appointment / staff / billing for salons)

**Brand vibe:** soft, friendly, beauty-tone, modern.

**Typical assets needed:**
- Salon interior photos — Unsplash / Pexels "salon", "spa",
  "haircut" — flag model release for identifiable people.
- Appointment / calendar illustrations — Storyset, unDraw.
- Icons for appointment, staff, services, billing — Lucide / Tabler
  (rounded weight fits beauty brands).
- Beauty / lifestyle short video clips for Instagram reels (Pexels
  Videos, Mixkit) — flag model release.
- Lottie animations for "appointment booked", "payment received".

**Avoid:** stock photos that look heavily filtered or dated.

## BRC-Buddy (children / therapy support)

**Brand vibe:** warm, child-friendly, safe, calm. Soft colours, rounded.

**Typical assets needed:**
- **Illustrations only** for children — Storyset "kids / school /
  therapy" packs, Open Doodles, unDraw with friendly characters.
- Therapy / activity icons — Phosphor (rounded), Lucide.
- Safe avatars — Boring Avatars, DiceBear (lorelei / fun-emoji /
  pixel-art).
- Lottie animations for cheerful interactions, rewards, progress.

**Avoid (hard rule):** real-child photos, identifiable faces, anything
that could be misused. Default to illustration-only. Flag any photo
candidate in risk notes.

## Trivasia (travel)

**Brand vibe:** aspirational, vibrant, real-world, photographic.

**Typical assets needed:**
- Destination photos — Unsplash, Pexels "Goa", "Kerala", "Himachal",
  "Bali", "Paris" etc.
- Travel illustrations / map graphics — Storyset, SVG Repo.
- Icons for flight, hotel, package, weather, map — Lucide / Tabler /
  Phosphor.
- Short video clips for destinations — Pexels Videos, Coverr, Pixabay
  Videos.
- Weather Lottie animations — LottieFiles "weather / sun / rain".

**Flag:** property release for recognisable hotels / landmarks; for
ad use, verify per-photo licence and release.

## Social media assets (all projects)

**Aspects:**
- Reels / Stories: 1080×1920 (9:16).
- Posts: 1080×1080 (1:1).
- Carousel: 1080×1350 (4:5).
- YouTube thumbnail: 1280×720 (16:9).
- LinkedIn post: 1200×627.

**Typical assets needed:**
- Backgrounds (textures, gradients) — Pixabay, SVG Repo.
- Icon stickers — Lucide / Tabler.
- Motion graphics / Lottie loops.
- Short b-roll clips (Pexels Videos, Coverr, Mixkit).
- Music / SFX — Pixabay Music, YouTube Audio Library (verify per
  track).

**Flag:** GIPHY watermarked stickers for any paid ad creative.

## Default LeadUp picks per asset type

| Asset type | Default source(s) | Why |
|---|---|---|
| Photo | Unsplash → Pexels | No-attribution + commercial. |
| Illustration | unDraw → Open Doodles → Storyset (if attribution OK) | LeadUp's flat style. |
| Icon set | Lucide → Tabler → Heroicons | One set per product. |
| Lottie | LottieFiles (Free + Commercial filter) | Largest curated free pool. |
| GIF (messaging) | Pixabay GIFs | No baked watermark. |
| 3D model | Sketchfab CC0/CC-BY → Spline Community | Best free quality. |
| Avatar | DiceBear → Boring Avatars | Seedable, MIT. |
| Video clip | Pexels Videos → Coverr | Clean licence. |
| Music | Pixabay Music → YouTube Audio Library | Verify per track. |
| Emoji | Native Unicode → Noto | No attribution burden. |
