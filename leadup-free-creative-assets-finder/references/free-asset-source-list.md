# Free Asset Source List

Curated list of free / safe-to-use creative-asset sources used by LeadUp.
For every source, capture the licence summary, attribution rule, and any
restrictions. Always verify against the official terms on the source site
before recommending — terms change.

> Notation in the "License (typical)" column is a summary, not legal
> advice. The official terms page on the source itself is the source of
> truth. Where any item is ambiguous, mark it **Needs license
> verification**.

---

## 1. Photos / stock images

| Source | License (typical) | Attribution | Commercial | Notes |
|---|---|---|---|---|
| Unsplash | Unsplash License | Not required (appreciated) | Yes | No resale of photo as-is. No use in identifiable trademark. Model release not guaranteed. |
| Pexels | Pexels License | Not required (appreciated) | Yes | No resale unmodified. No use to defame people. No use in trademarks. |
| Pixabay | Pixabay Content License | Not required | Yes | No identifiable people in commercial unless release; no trademarks; no resale unmodified. |
| Wikimedia Commons | Per-file (CC0 / CC-BY / CC-BY-SA / PD) | **Per-file**, often required | Yes (per-file) | License varies file by file — read each file's page. |
| Openverse | Aggregator: CC0 / CC-BY / CC-BY-SA / PDM | **Per-file** | Per-file | Openverse only indexes; verify on the originating source. |

Search priority for LeadUp: Unsplash → Pexels → Pixabay → Openverse (for
CC0/CC-BY filter) → Wikimedia (for historical, scientific, geographic).

## 2. Videos / video clips

| Source | License (typical) | Attribution | Commercial | Notes |
|---|---|---|---|---|
| Pexels Videos | Pexels License | Not required | Yes | Same as Pexels photos. |
| Pixabay Videos | Pixabay Content License | Not required | Yes | Same as Pixabay photos. |
| Coverr | Coverr License | Not required | Yes | Designed for website background loops. |
| Mixkit | Mixkit License | Not required (per asset) | Yes (per asset) | Some items "free for personal" only — check per asset. |
| Videvo | Mixed (Videvo Standard / CC-BY / paid) | Per-file | Per-file | Filter to "Free" + "Commercial" carefully. |

Prefer Pexels Videos and Coverr for hero loops; Pixabay for short clips;
Mixkit / Videvo with per-asset verification.

## 3. Illustrations / vectors

| Source | License (typical) | Attribution | Commercial | Notes |
|---|---|---|---|---|
| unDraw | unDraw License (MIT-style) | Not required | Yes | Free recolour; do not resell as-is. |
| Storyset (Freepik) | Free for personal & commercial **with attribution** unless paid plan | **Required on free tier** | Yes | Attribution required on free tier — flag for client work. |
| SVG Repo | Mixed (MIT / CC0 / CC-BY / Logo) | Per-file | Per-file | License shown per icon — verify before use. |
| Open Doodles | CC0 | Not required | Yes | Hand-drawn doodle style. |

For LeadUp brand-fit: unDraw + Open Doodles (no-attribution) → Storyset
(only if attribution can be shown, or on paid plan).

including SaaS UI.

|---|---|---|---|
| Iconify | Aggregator — per icon set | SVG (via API / components) | License of underlying set applies. |
| Lucide | ISC | SVG, React, Vue, Svelte | LeadUp default for SaaS UI. |
| Heroicons | MIT | SVG, React, Vue | By Tailwind team. |
| Tabler Icons | MIT | SVG, React, Vue, Svelte | 5400+ icons. |
| Feather Icons | MIT | SVG | Minimal line set. |
| Remix Icon | Apache 2.0 | SVG | Large set, multiple styles. |
| Phosphor Icons | MIT | SVG, React, Vue | Multiple weights. |

icon-weight matches.

## 8. Emojis

| Source | License | Notes |
|---|---|---|
| Native Unicode emoji | Free, OS-rendered | No payload; renders per OS. |
| Twemoji | CC-BY 4.0 (graphics) | Attribution required to Twitter/Twemoji team. |
| OpenMoji | CC-BY-SA 4.0 | Attribution + share-alike required. |
| Noto Emoji | OFL / Apache 2.0 | Google's set; permissive. |

For client work where attribution is hard to place, native Unicode or
Noto Emoji are safest. Twemoji and OpenMoji **require attribution**.

## 9. GIFs / stickers

| Source | License (typical) | Notes |
|---|---|---|
| Pixabay GIFs | Pixabay Content License | Safest for commercial. |
| GIPHY | Per-file; GIPHY watermark may be baked in | Watermarked stickers risky for paid client work. Brand-channel content is owned by brand. |
| Tenor | Per-file (Tenor / Google) | Verify each asset. |
| LottieFiles GIF exports | Same as the source Lottie file | Use the underlying Lottie licence. |

For client websites: prefer Lottie over GIF where possible. For
WhatsApp / messaging assets, GIFs are fine but verify watermark + brand
rules.

## 10. Sound effects / music (only when asked)

| Source | License | Notes |
|---|---|---|
| Pixabay Music | Pixabay Content License | Free for commercial; no attribution. Verify any vocal tracks. |
| Free Music Archive (FMA) | Per-track (CC variants / proprietary) | Filter to CC-BY / CC0 for commercial. |
| YouTube Audio Library | Per-track (royalty-free + CC-BY) | Read per track; some require attribution. |
| Freesound | Per-sound (CC0 / CC-BY / Sampling+) | Filter strictly. |

Always confirm both the audio licence **and** any platform rights (e.g.
YouTube Content ID) before recommending for ads / reels.

## Source-selection priority cheatsheet

- Photos → Unsplash, Pexels (no attribution, commercial OK).
- Videos → Pexels Videos, Coverr.
- Illustrations → unDraw, Open Doodles (no attribution); Storyset if
  attribution can be shown.
- Icons → Lucide / Heroicons / Tabler / Phosphor (MIT). Stick to one.
- Animations → LottieFiles (Free + Commercial filter).
- 3D → Sketchfab CC0/CC-BY, Spline Community, Pixabay 3D.
- Avatars → DiceBear, Boring Avatars.
- Emojis → native Unicode first.
- GIFs → Pixabay GIFs (avoid watermarked GIPHY stickers on paid surfaces).
- Music → Pixabay Music, YouTube Audio Library.

For any source not on this list, run the full
`license-checklist.md` check before recommending.
