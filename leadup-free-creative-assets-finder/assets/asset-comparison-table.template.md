# Asset Comparison Table

Use this as the candidate sources table inside an asset brief. Score
each candidate on the 8-criterion framework (each /5, total /40). Top
pick must score ≥ 4 on visual fit, license clarity, commercial safety,
and brand fit.

## Scoring rubric (each /5)

- **Visual fit** — matches the required style, mood, palette.
- **License clarity** — explicit, easy-to-read commercial terms.
- **Commercial safety** — safe for paid client work (attribution,
  redistribution, trademarks, releases all accounted for).
- **Format availability** — SVG / PNG / WebP / AVIF / Lottie / MP4 /
  WebM / GLB as needed.
- **Performance impact** — file weight, load strategy, mobile cost.
- **Customization ability** — recolour, resize, edit, animate.
- **Implementation ease** — drop-in component library, CDN, copy-paste
  embed.
- **Brand fit** — matches LeadUp / target client brand.

## Decision rules

- Top pick must score **≥ 4** on visual fit, license clarity, commercial
  safety, and brand fit.
- If no candidate scores ≥ 4 on commercial safety → **do not
  recommend**; output a "Needs license verification" entry instead.
- Tiebreaker order: license clarity → commercial safety → format
  availability → performance → visual fit.
