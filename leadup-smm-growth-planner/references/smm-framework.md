# SMM growth framework — LeadUp

How LeadUp plans social media growth for client businesses and our own
SaaS. Practical, low-overhead, and honest about numbers.

## 1. Platform fit (do not be on every platform)

Pick 1–3 platforms based on **audience × capacity**:

| Audience | Default mix |
|---|---|
| Indian local services (salons, clinics, restaurants) | Instagram + WhatsApp + Google Business |
| Indian D2C / ecommerce | Instagram + YouTube Shorts + WhatsApp |
| Indian SMB B2B (CRMs, billing software) | LinkedIn + Instagram + YouTube |
| Global B2B SaaS | LinkedIn + X + YouTube + (Reddit, niche) |
| Global D2C | Instagram + TikTok + YouTube Shorts |
| Education / coaching | YouTube Shorts + Instagram + LinkedIn |

State **why** for each platform you keep.

## 2. Content pillars (4–6, with a clear % each)

Default pillar set for a service business:

| Pillar | Share | Purpose |
|---|---|---|
| Education | 30% | rank as the expert |
| Behind-the-scenes | 20% | trust + humanity |
| Social proof | 20% | results, reviews, before/after |
| Offers / launches | 15% | direct CTA |
| Community / UGC | 10% | conversation, reach |
| Founder voice | 5% | brand authority |

For SaaS, swap "behind-the-scenes" for "product updates" and increase
"education" to 40%.

Lock the pillar mix **before** drafting the calendar.

## 3. Hooks (the first 1–3 seconds)

Hook patterns that work, used sparingly:

- **Pattern interrupt** ("Stop doing X. Here's why.")
- **Specific outcome** ("How we cut no-shows by 30% in one week.")
- **Insider POV** ("What 10 salon owners in Coimbatore told us.")
- **Mistake** ("3 mistakes that kill your booking rate.")
- **Number** ("7 questions every clinic owner should ask.")
- **Contradiction** ("Discounts don't grow your salon. This does.")
- **Local angle** ("Why tier-2 salons are different.")
- **Question** ("Why did one ad get 4× the leads?")

Rules:
- Hook within 6 words on Reels / Shorts.
- Avoid "Did you know" as a default.
- One hook per post, not three stacked.

## 4. Caption patterns

3–5 patterns to reuse:

1. **Story → Insight → CTA** (3–5 short paragraphs).
2. **List → CTA** (3–5 specific bullets).
3. **Contrarian → Reframe → CTA**.
4. **Behind-the-scenes → Lesson → CTA**.
5. **Founder POV → Question to audience**.

Caption length:
- Instagram: 60–150 words ideal; first 125 chars must hook.
- LinkedIn: 100–250 words; native, no link in first version (link in
  comments often performs better).
- X / Threads: ≤ 280 chars per post; use threads for depth.

CTAs: one per caption. Specific ("DM 'BOOK' to get the offer") beats
generic ("Check our website").

## 5. Posting cadence

Match cadence to the team's actual capacity, not industry "best practice":

| Capacity | Default per platform / week |
|---|---|
| 1 person, no editor | 2–3 posts |
| 1 person + part-time editor | 4–5 posts |
| Small team (2–3) | 5–7 posts + stories daily |

Reels / Shorts > static for new account growth.
Stories / status > main feed for retention + warm DMs.

## 6. Posting times (anchor to audience timezone)

For Indian audiences (IST), default starting windows:

| Day | Time band | Why |
|---|---|---|
| Mon–Fri | 8–10am IST | commute / morning scroll |
| Mon–Fri | 1–2pm IST | lunch |
| Mon–Fri | 7–10pm IST | prime evening scroll |
| Sat–Sun | 10am–12pm, 6–9pm IST | weekend cadence |

For B2B global: 8–10am and 5–7pm in the buyer's timezone. State the
timezone in the plan.

These are starting points; replace with the user's own analytics once
2–4 weeks of data exist.

## 7. Engagement plan

The first 60 minutes after posting matters more than the next 24 hours:

- Reply to every comment within the first 60 minutes.
- Open DMs from new followers within the first hour where possible.
- Pin one strong comment to anchor the conversation.
- Use stories to drive eyeballs to the post in the first 30 minutes.
- Save 10–15 minutes/day for outbound: comment on 5–10 niche posts.

Weekly rituals:

- Mon: plan + schedule the week.
- Wed: review analytics from the past week.
- Fri: respond to outstanding DMs.

DM scripts (kept short, human, not AI-templated):

```
Hi [name], saw your follow. We help [audience] with [outcome].
Are you exploring this for [their use case]? Happy to share a quick
1-minute video walkthrough.
```

## 8. Hashtags

- 5–10 hashtags per post on Instagram (not 30).
- Mix sizes: 1–2 large (millions), 4–6 medium (50K–500K), 2–3 small
  (under 50K) niche tags.
- For local: include 2 city/locality tags.
- LinkedIn: 3–5 hashtags inline.
- X: 1–2 hashtags max.
- Skip hashtag stuffing in the caption — looks AI-templated.

## 9. KPIs (4–6, with bands not invented numbers)

Default KPI set:

| KPI | Where to read | Why |
|---|---|---|
| Reach band (per pillar) | native analytics | which pillar pulls |
| Profile visits / week | native analytics | discovery funnel |
| Save rate | native analytics | content quality signal |
| Reply / DM rate | native + manual | warm intent |
| Booking / signup attributed | UTM + GA4 + CRM | actual revenue |
| Follower growth band | native analytics | trailing indicator |

Target as **bands** (e.g. "reach band: 5K–10K/post within 30 days,
assuming 4 posts/week and current follower base"), never as fixed
numbers.

## 10. Output

Always deliver:

- A platform strategy block (why these platforms).
- A pillar mix that sums to 100%.
- A 30-day calendar (markdown table + CSV asset).
- A hooks library + caption patterns.
- A posting schedule with stated timezone + reasoning.
- An engagement plan covering the first 60 minutes + weekly rituals.
- A KPI block with bands and "where to read".
- An assumptions block with verified / estimated / requires verification.

Defer caption polish to `leadup-human-content-editor`.
