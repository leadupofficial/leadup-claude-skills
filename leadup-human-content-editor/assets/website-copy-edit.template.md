# Website copy edit — fill-in template

Use this template when humanising a landing page, service page, or about
page. Keep all existing headings, sections, and CTAs. Rewrite the body
text into LeadUp's natural, business-specific voice.

---

## Inputs

- **Page type:** landing / service / about / pricing / contact / [other]
- **Audience:** [who reads this page — industry, role, end customer]
- **Brand voice already in use on the site:** [formal / casual / mixed]
- **Must-keep items:** brand name, product name, SEO keywords, CTAs,
  numbers, client names.
- **Banned extras:** [any words the user wants to avoid]
- **Word limit per section:** [optional]

---

## Mode + assumptions

> Mode: Website landing page · Tone: simple business (default)
> Assumption: [one line — only if the source had unclear intent]

---

## Rewritten content

````markdown
# [H1 — keep or rewrite to be specific and short]

[Sub-hero — 1–2 sentences. Name the audience. Name the outcome.
No em dashes. No hype words.]

[CTA button text — kept from source]

## [Section H2 — short, specific]

[2–4 sentences. One idea per sentence. Mix sentence lengths.]

- [Bullet — concrete deliverable or feature, varying length]
- [Bullet — a different verb shape from the first]
- [Bullet — add a number or specific where the source has one]

## [Next H2…]

[…]

[Closing CTA — single, clear action. No pep talk.]
````
