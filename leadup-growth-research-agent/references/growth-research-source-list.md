# Growth research source list — LeadUp

The sources LeadUp consults when doing growth research. Always cite the
source name + access date for any claim that isn't general knowledge.

## 1. Search

- **Google** — the SERP itself, "People also ask", related searches,
  autocomplete suggestions, image / video tabs, Maps for local intent.
- **Bing / DuckDuckGo** — sanity check for SERP differences and
  alternative results.
- **YouTube search + autocomplete** — what users phrase as video queries.

## 2. SEO + technical

- **Google Search Console** (if the user has access) — queries, CTR,
  coverage, Core Web Vitals report, mobile usability.
- **Google Keyword Planner** — volume bands + competition (for ads, but
  usable as SEO hints).
- **Google Trends** — seasonality and rising terms by country.
- **PageSpeed Insights / Lighthouse** — Core Web Vitals per page.
- **Google Search Central docs** — official guidance on indexing,
  rendering, structured data.
- **Schema.org** — structured-data definitions.
- **web.dev** — performance, accessibility, PWA guidance.
- **Bing Webmaster Tools** — secondary signal.
- **Ahrefs / Semrush / Moz** — only if the user has access; cite the
  tool.

## 3. Social

- **Instagram Explore + Reels Top** — trending hooks and formats by
  niche.
- **LinkedIn search** — top posts and creators in the role-based niche.
- **YouTube Shorts trending** — short-form formats winning now.
- **X / Twitter search** — sentiment and breaking conversations.
- **Reddit** — niche subreddits for user pain points and language.
- **Quora** — long-tail questions and answers.
- **TikTok search + Creative Center** — discovery and ad inspiration
  where the user is on TikTok.

## 4. Ads

- **Meta Ad Library** — active ads, creative, copy, landing pages of
  competitors.
- **Google Ads Transparency Center** — competitor Search and Display
  ads.
- **LinkedIn Ad Library** — competitor B2B creative.
- **YouTube** — competitor video ads by search.
- **Apple Search Ads** — App Store ad inspiration if the user is on
  iOS.

## 5. Analytics + UX (if user has access)

- **GA4** — channels, conversion paths, landing pages, retention.
- **Microsoft Clarity** — heatmaps + recordings (free).
- **Hotjar / FullStory** — heatmaps + recordings (paid).
- **Form analytics** — field interactions, abandonment.

## 6. Competitor sites (public only)

- Homepage, services / pricing / about pages.
- `sitemap.xml` and `/blog/` index for topic coverage.
- Changelog or "what's new" for feature signal.
- LinkedIn company page for hiring + funding cues.

## 7. Communities + reviews

- **G2 / Capterra** — SaaS reviews + comparisons.
- **ProductHunt** — launches in the category.
- **Trustpilot** — service business reviews where relevant.
- **Reddit** + **niche forums** — voice of customer.
- **Indie Hackers** — small-SaaS learnings.

## 8. India-specific (regulation + payments)

- **ASCI** — advertising code of conduct.
- **IT Rules 2021 (Intermediary Guidelines + Digital Media Ethics)** —
  for content and platform claims.
- **DPDP Act** (when in scope) — data protection.
- **GST rules** — invoicing, tax invoices, reverse charge.
- **Razorpay / Cashfree / Stripe-India docs** — payments + KYC.
- **RBI guidance** — for fintech / lending / wallets.
- **TRAI guidance** — for SMS / DLT templates.
- **WhatsApp Business / Meta BSP docs** — template approvals.

## 9. Global compliance

- **GDPR / EU AI Act** when serving EU users.
- **CCPA / state privacy laws** when serving US users.
- **PCI DSS** when handling card data.
- **SOC 2** for B2B SaaS prospects.

## 10. AI / model documentation

- **Anthropic docs** — Claude API, prompt caching, citations, tool use.
- **OpenAI docs** — model + API reference (if user is on OpenAI).
- **Google Gemini docs** — Gemini API.
- **Hugging Face** — open models.
- **Vector DB docs** — Pinecone / pgvector / Qdrant / Weaviate.

## 11. How to cite

Every claim that isn't general knowledge should look like:

> "Top competitor uses a comparison table on their pricing page (source:
> competitor.com/pricing, accessed YYYY-MM-DD)."

Or:

> "Estimated CPL band ₹X–Y for the category (estimated; requires
> verification via Meta Ads Manager on user's account)."

## 12. What NOT to use

- Scraped paid-content (paywalls, gated reports) without licence.
- Auto-generated "AI summary" pages that aren't primary sources.
- Single forum posts as the only signal for a major decision.
- Numbers without a source — if you can't cite it, mark **estimated**.
